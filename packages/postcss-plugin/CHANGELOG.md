## 3.0.0 - 2020-03-28

This package has been renamed from `postcss-shopify` to `@shopify/postcss-plugin.`. Update any configuration to use this new name

Before:

```
module.exports = {
  plugins: {
    'postcss-shopify',
  },
};
```

After:

```
module.exports = {
  plugins: {
    '@shopify/postcss-plugin',
  },
};
```
